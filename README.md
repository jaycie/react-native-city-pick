# react-native-city-pick

#### 采用picker选择，兼容android , ios

#### 下载组件

```
$ git clone https://github.com/jaycie/react-native-city-pick/
```

#### 运行

```

$ cd react-native-city-pick
$ npm install
$ react-native run-ios
```
